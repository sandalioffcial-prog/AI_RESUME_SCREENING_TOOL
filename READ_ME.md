AI Resume Screening Tool
Project Overview

The AI Resume Screening Tool is a machine learning and NLP-based application that automates the resume screening process. It analyzes resumes, extracts relevant information, compares candidate profiles with job requirements, and helps recruiters shortlist suitable candidates.

Objectives

:Automate resume screening.
:Reduce recruitment time.
:Improve candidate selection accuracy.
:Rank resumes based on job requirements.

Technologies Used

:Python
:Pandas
:NumPy
:Scikit-learn
:NLTK
:Streamlit

Features

:Resume Upload
:Resume Parsing
:Skill Extraction
:Candidate Ranking
:Job Description Matching

Project Workflow

:Upload Resume
:Extract Resume Text
:Text Preprocessing
:Feature Extraction
:Similarity Calculation
:Candidate Ranking
